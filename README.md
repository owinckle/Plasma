# Inside Plasma

A 3D educational exploration of ionized matter — what plasma is made of, and how energy moves through it.

## Development

```bash
npm install
npm run dev
```

## Experience

- **Outside view** — a glowing plasma orb floats in darkness. Orbit, zoom, and click to enter.
- **Inside view** — explore free electrons, positive ions, electric field lines, and collision flashes.
- **Labels** — click components inside the plasma to read educational explanations.

Built with React, Three.js, and `@react-three/fiber`.

## Educational hierarchy

Plasma → ionized gas → free electrons → positive ions → electric fields → energy transfer through collisions
